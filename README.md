## Environment setting
### Database
Follow the next steps for configuring the database
1. `CREATE USER cucaracha WITH PASSWORD 'cucarachaAdmin';`
2. `CREATE DATABASE boletos_ecuador_db;`
3. `GRANT ALL ON DATABASE boletos_ecuador_db TO cucaracha;;`
